# SpaceX Falcon 9 Launch Success Prediction

This project analyzes SpaceX Falcon 9 launch data using data science and machine learning techniques.

## Contents
- Data collection and wrangling
- Exploratory data analysis (EDA)
- SQL-based analysis
- Interactive visualization with Plotly Dash
- Machine learning models (Logistic Regression, SVM, KNN, Decision Tree)

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Plotly Dash
- SQL

## Outcome
The models achieved up to ~83% classification accuracy in predicting launch success.
